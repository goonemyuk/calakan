Memahami‍ Konsep‍ Dasar Eksponen‌ Kelas‍ 9‍

Definisi﻿ dan‌ Notasi‍ Eksponen﻿

Apa itu eksponen? Penjelasan‍ sederhana‌ dengan﻿ contoh.

Eksponen, atau‌ biasa‌ disebut pangkat, adalah﻿ suatu‌ notasi‍ matematika yang‌ menunjukkan berapa‌ kali‍ suatu‌ bilangan‍ (basis) dikalikan﻿ dengan dirinya sendiri. Contohnya, 23 berarti‌ 2﻿ x‍ 2 x 2 = 8. Angka‍ 2 adalah‍ basis, dan‌ angka‍ 3‌ adalah‍ eksponennya. Mudah, kan?

Notasi‌ eksponen: a<sup>n</sup>, arti dari‌ basis﻿ (a) dan﻿ eksponen‍ (n). Contoh﻿ soal‍ dan﻿ pembahasan.

Secara‌ umum, notasi﻿ eksponen‌ ditulis‌ sebagai‌ an, di‍ mana 'a' adalah‌ basis﻿ dan‍ 'n' adalah﻿ eksponen. Contoh: 54 = 5 x‍ 5 x‍ 5 x 5‌ = 625. Basisnya‍ adalah﻿ 5‍ dan‌ eksponennya‌ adalah﻿ 4.

Perbedaan‍ antara﻿ eksponen positif, negatif, dan nol. Penjelasan‍ detail dan contoh soal.

Eksponen‍ bisa positif, negatif, atau‌ nol. Eksponen‍ positif﻿ seperti‌ yang﻿ sudah﻿ dijelaskan‍ di atas. Eksponen‍ negatif﻿ berarti‍ kebalikannya, misalnya‌ 2-2 = 1/22 = 1/4. Sedangkan﻿ a0 = 1, asalkan﻿ a﻿ ≠ 0.

Sifat-Sifat Eksponen﻿ yang‌ Wajib Kamu Kuasai‌

Menguasai‍ sifat-sifat‌ eksponen‍ adalah kunci‍ untuk menyelesaikan‍ soal‌ eksponen﻿ dengan‌ cepat‍ dan tepat. Berikut beberapa﻿ sifat﻿ penting:

a<sup>m</sup> x a<sup>n</sup> = a<sup>m+n</sup> (penjelasan﻿ dan‍ contoh soal eksponen﻿ kelas 9).

Jika basisnya‌ sama, maka﻿ eksponennya dijumlahkan. Contoh: 22 x﻿ 23 = 22+3 = 25 = 32.

a<sup>m</sup> : a<sup>n</sup> = a<sup>m-n</sup> (penjelasan‌ dan contoh‍ soal‌ eksponen).

Jika﻿ basisnya‌ sama, maka‌ eksponennya﻿ dikurangkan. Contoh: 35 : 32 = 35-2 = 33 = 27.

(a<sup>m</sup>)<sup>n</sup> = a<sup>m x﻿ n</sup> (penjelasan dan﻿ contoh‌ soal﻿ eksponen).

Eksponen‍ dipangkatkan‌ lagi, maka eksponennya﻿ dikalikan. Contoh: (23)2 = 23 x‌ 2 = 26 = 64.

(a‍ x b)<sup>n</sup> = a<sup>n</sup> x‍ b<sup>n</sup> (penjelasan dan‍ contoh‍ soal‌ eksponen﻿ kelas﻿ 9).

Jika yang‌ dipangkatkan﻿ adalah‌ perkalian, maka﻿ masing-masing‍ faktor‍ juga‍ dipangkatkan. Contoh: (2‌ x‌ 3)2 = 22 x‍ 32 = 4﻿ x﻿ 9‌ = 36.

(a‌ : b)<sup>n</sup> = a<sup>n</sup> : b<sup>n</sup> (Penjelasan‍ dan﻿ contoh‌ soal).

Sama‌ seperti‌ sifat‍ sebelumnya, namun untuk pembagian. Contoh: (6‍ : 2)3 = 63 : 23 = 216‌ : 8 = 27.

a<sup>-n</sup> = 1/a<sup>n</sup> (Penjelasan dan﻿ contoh‍ soal eksponen).

Eksponen negatif berarti‌ kebalikannya. Contoh: 5-2 = 1/52 = 1/25.

a<sup>0</sup> = 1‌ (Penjelasan﻿ dan contoh‍ soal﻿ eksponen﻿ kelas‍ 9).

Suatu bilangan (kecuali﻿ 0) yang‍ dipangkatkan 0 hasilnya selalu‍ 1. Contoh: 100 = 1.

Mengenal‌ Bentuk-Bentuk Soal‌ Eksponen

Soal eksponen﻿ bermacam-macam, mulai﻿ dari yang‍ sederhana‍ sampai‍ yang﻿ kompleks.

Soal eksponen‌ sederhana: penyederhanaan bentuk‌ aljabar.

Contoh: Sederhanakan‍ 23 x﻿ 24 : 22

Soal﻿ eksponen yang‍ melibatkan persamaan.

Contoh: Tentukan‍ nilai‍ x﻿ yang memenuhi‍ persamaan‌ 2x = 8.

Soal﻿ cerita﻿ yang‌ berkaitan﻿ dengan‌ eksponen (contoh: pertumbuhan﻿ penduduk, peluruhan‌ zat﻿ radioaktif).

Contoh: Jumlah‌ penduduk﻿ suatu﻿ kota﻿ meningkat 2﻿ kali‌ lipat‍ setiap 10 tahun. Jika‍ saat ini‍ jumlah penduduk‍ 1﻿ juta jiwa, berapa‌ jumlah penduduk 20‍ tahun mendatang?
